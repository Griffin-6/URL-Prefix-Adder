# URL Prefix Adder – Chrome 扩展

一键为当前网址添加自定义前缀，适用于视频换线路、镜像站访问、测试环境跳转等场景。

## 功能

- 悬浮工具栏（返回顶部、收藏、历史、换线、打开）
- 前缀管理：增删改查、备注、导入导出 txt / JSON
- 历史只记录插件打开的页面（不污染正常浏览）
- 深浅主题、右键隐藏悬浮球

## 安装

1. 下载本仓库代码（ZIP 解压 或 `git clone`）
2. Chrome 打开 `chrome://extensions/`，开启“开发者模式”
3. 点击“加载已解压的扩展程序”，选择文件夹

## 使用

- 点击扩展图标 → 添加前缀（必须以 http:// 或 https:// 开头）
- 网页右侧悬浮球 → 展开工具栏 → 换线 / 打开
- 右键扩展图标可临时隐藏悬浮球

## 技术栈

原生 JavaScript + Manifest V3 + Chrome Storage API，零依赖。

## 注意

前缀内容由用户自行定义，使用后果自负。
